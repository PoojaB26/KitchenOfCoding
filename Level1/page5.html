When you are reading a recipe, there are few footnotes or comments or instructions before the main recipe. In the same way, if we want to add a comment or instruction to our code, we can use # to write anything we want. We call it a comment.

When somebody reads your recipe, these sidenotes help other readers to understand your recipe better.
In the same way when someone reads your code, these comments help them to understand what you are trying to do in the code.
Let’s show you an example of how comments are used in coding
