Before we proceed, let me ask you,
why do we use functions in the first place?

Lines of code that can be executed as a unit is called a code block.
When you call a function, all the lines of code defined in a function are automatically called. So a function defines a code block. And to define a code block, we use indentation to separate it from the rest of the lines of code.

Let's start redefining our boil() function.

def marks the start of the function header. We write the name of the function next. You can call your function anything as long as it correctly defines the purpose of the function.

Fun fact : Function names are mostly verbs.

End the function header with a :

Any code with indentation under the function header is part of the function now.
